# Harshit Goswami — AI/ML Portfolio Website

A modern, responsive, and visually immersive personal portfolio website built for showcasing AI/ML projects, research experience, technical skills, and developer activity.  
This portfolio combines futuristic UI design with interactive animations, project showcases, GitHub activity tracking, AI demo interactions, and responsive layouts to create a strong professional presence online.

---

## Features

- Modern futuristic UI/UX design
- Fully responsive across devices
- Interactive hero section with animated visuals
- AI/ML focused project showcase
- GitHub activity visualization
- Research & experience sections
- Dynamic project filtering system
- AI chatbot demo interface
- Smooth scrolling and reveal animations
- Custom preloader animation
- Professional contact form
- SEO optimized metadata
- Lightweight and fast-loading structure
